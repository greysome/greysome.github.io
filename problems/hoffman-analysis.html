1.2.1: Set of rational numbers not bounded below

Is the set of rational numbers bounded below?

No. If it is, then let \(x = \inf \Q\). By definition, \(x \le r\) for every rational number \(r\). Since \(r-1\) is also a rational number, this means \(x \le r-1\) and so \(x+1 \le r\), which implies \(x+1\) is a lower bound for \(\Q\). This contradicts the fact that \(x\) is the greatest lower bound.

1.2.3: Non-empty bounded sets where supremum \(\le\) infimum

Find all non-empty bounded sets \(A\) such that \(\sup A \le \inf A\).

The only such sets are \(\{r\}\) where \(r \in \R\): in this case \(\sup A = \inf A\). For if there are two points \(x,y \in A\) such that \(x < y\), then by definition it follows that \[ \inf A \le x < y \le \sup A. \]

1.2.4: Boundedness of the empty set

Is the empty set bounded above? Does it have a least upper bound?

Any real number \(r\) is an upper bound for \(\varnothing\): it is true that \(x \le r\) for all \(x \in \varnothing\), since there is no such \(x\). However, since the set of upper bounds is precisely \(\R\), there is clearly no least upper bound.

1.2.7: Real numbers as supremums

Prove that, if \(x\) is a real number, then \[x = \sup \{r \in \Q \mid r < x\}.\]

Let \(A = \{r \in \Q \mid r < x\}\). By definition, \(x\) is an upper bound of \(A\). Suppose then that \(x\) is not a least upper bound: there exists some \(x' \in \R\) such that \(x' < x\), and that \(r \le x'\) whenever \(r \in A\) (i.e. \(r < x\)). This is a contradiction, because there exists a rational number \(r'\) such that \(x' < r' < x\).

1.2.8: Irrationals are dense in the reals

If \(x < y\), there exists an irrational number \(t\) such that \(x < t < y\).

Suppose \(x\) is irrational. There is a positive integer \(n\) such that \(n > 1/(y-x)\). Since \(y-x > 0\), it follows that \(n(y-x) = ny-nx > 1\). Hence \(ny > nx+1 > nx\) so \(y > (nx+1)/n > x\); we can then let \(t = (nx+1)/n\) since it is irrational.

Suppose \(x\) is rational. Likewise, there is a positive integer \(n\) such that \(n(y-x) = ny-nx > \sqrt{2}\). Hence \(ny > nx+\sqrt{2} > nx\) so \(y > (nx+\sqrt{2})/n > x\); we can let \(t = (nx+\sqrt{2})/n\) since it is irrational.

1.2.9: Non-empty set of positive integers contains its infimum

Verify that every non-empty set of positive integers contains its infimum.

Let \(A\) be a non-empty set of positive integers; then \(A\) is bounded below by \(0\). Thus the set \[B = \{n \in \Z \mid n \ge 0\ \text{and $n$ is a lower bound of $A$}\}\] is non-empty. Furthermore, \(B\) must be finite, otherwise it would not be bounded above. This would yield a contradiction: for any real number \(r\), there would be some \(b \in B\) such that \(r < b\). Fixing some \(a \in A\), it follows that \(r < b \le a\) for all \(r \in \R\), which is absurd.

Let \(a,b\) be integers. If \(a < b\), then \(a+1 \le b\).

Suppose \(a < b\) and \(a+1 > b\). Then we have \(b-1 < a < b\), which is clearly a contradiction, since there cannot be an integer between 2 consecutive integers.

Since \(B\) is non-empty and finite, it has a maximum element \(x\). By definition, \(x \le a\) for all \(a \in A\). In fact, \(x = y\) for some \(y \in A\), otherwise by the lemma \(x+1\) would also be a lower bound for \(A\) and hence an element of \(B\), contradicting the maximality of \(x\). It can then be easily seen that \(y = \inf A\), completing the proof.

1.2.10: Subset of uncountable set not containing its supremum

Let \(A\) be an uncountable subset of \(\R\). Prove that there exists a subset \(B \subset A\) such that \(\sup B \not\in B\).

Suppose \(A\) is a non-empty set of integers that is bounded above (below). Then \(\sup A\) (\(\inf A\)) is an integer.

Let \(r\) be an upper bound of \(A\). By the Archimedean property, there is a positive integer \(x\) such that \(x > r\), so that \(x\) is also an upper bound. Using the same argument as in 1.2.9, it can be concluded that the set \[B = \{n \in \Z \mid n \le x\ \text{and $n$ is an upper bound of $A$}\}\] is non-empty and finite, and hence contains a maximum element \(b\). Then it can be proven that \(b = \sup A\), so that \(\sup A\) is an integer. The argument for \(\inf A\) is essentially the same.

Let \(r\) be a real number. Then there is some integer \(x\) such that \(x \le r < x+1\).

Let \(A = \{n \in \Z \mid n \le r\}\) and \(x = \sup A\). By the previous lemma, \(x\) is an integer. Also, we have \(x \le r\), because if \(x > r\) then there exists a smaller upper bound for \(A\), like \((x+r)/2\). Lastly, \(r < x+1\) because otherwise \(x+1\) is in \(A\) which implies \(x+1 \le x\).

Let \(A_n = A \cap [n,n+1)\). The second lemma then says that \(A\) is the union of the countable family \(\{A_n\}_{n \in \Z}\). Since \(A\) is uncountable, it follows that some \(A_k\) is uncountable.

Now construct a sequence of sets as follows:

\begin{align*} B_0 &= A_k \\ B_{i+1} &= B_i \setminus \{\sup B_i\}. \end{align*}

Note that \(\sup B_i\) exists since each \(B_i\) is bounded above by \(n+1\).

1.2.11: Existence of least upper bound implies completeness

Prove the completeness of the reals from Theorem 1 (a non-empty subset of \(A \subset \R\) that is bounded above has a least upper bound).

Let \(S,T\) be subsets of \(\R\) satisfying the hypotheses of the completeness property. Any element of \(T\) is an upper bound for \(S\), hence \(S\) has a least upper bound \(x\), which must be in either \(S\) or \(T\). If \(x \in S\), then \(x\) is the greatest element of \(S\) since \(x \ge s\) for all \(s \in S\). If \(x \in T\), then \(x\) is the least element of \(T\) since \(x \le t\) for all upper bounds \(t\).

1.2.12: \(\R\) contains no complete ordered proper subfield

Prove that if a subset of \(S\) of \(\R\) (with the inherited addition, multiplication, and ordering) is a complete ordered field, then \(S = \R\).

Since \(S\) is a subfield of the characteristic \(0\) field \(\R\), it must contain the prime subfield of \(\R\), namely \(\Q\). From 1.2.7, each real number \(x\) can be identified as the supremum of the set \[A = \{r \in \Q \mid r < x\}.\]

Clearly \(A \subseteq S\), and there is a positive integer \(n\) such that \(n > x\), so \(A\) is bounded above by an element of \(S\). Since \(S\) is complete, \(A\) must therefore have a least upper bound \(x' \in S\). It then remains to show that \(x = x'\). If \(x' < x\), then \(x\) is not a least upper bound of \(A\) in \(\R\); if \(x < x'\), then there is a rational number \(r \in S\) such that \(x < r < x'\), hence \(x'\) is not a least upper bound of \(A\) in \(S\). Thus \(x = x' \in S\), and this completes the proof.

1.2.13: Any two complete ordered fields are isomorphic

Let \(R\) and \(S\) be complete ordered fields. Show that \(R\) and \(S\) are isomorphic.

Let \(0_F\) and \(1_F\) denote the additive and multiplicative identitites of the field \(F\). Firstly, \(R\) and \(S\) must have characteristic \(0\): if \(n = 0_F\) for some positive integer \(n\), then it follows that \[0_F <1_F < 1_F+1_F < \ldots < \underbrace{1_F+\ldots+1_F}_\text{$n$ times} = n = 0_F,\] which is a contradiction. Thus, their prime subfields are isomorphic to \(\Q\).

Let \(\Q_F\) denote the isomorphic copy of \(\Q\) in \(F\). Then there is an isomorphism \(\varphi : \Q_R \rightarrow \Q_S\) mapping \(0_R \mapsto 0_S\) and \(1_R \mapsto 1_S\). We can see that \(\varphi\) also preserves the order of rationals: clearly it preserve the order of integers, so that if \(a,b,c,d\) are integers in \(R\) such that \(ad < bc\), then \[\varphi(a)\varphi(d) = \varphi(ad) < \varphi(bc) = \varphi(b)\varphi(c).\]

Expressing the previous statement in terms of fractions, if \(a/b < c/d\), then \[\varphi\left(\frac{a}{b}\right) = \frac{\varphi(a)}{\varphi(b)} < \frac{\varphi(c)}{\varphi(d)} = \varphi\left(\frac{c}{d}\right).\]

By 1.2.7, each element \(x\) of \(R\) can be identified as the supremum of the set \[A_x = \{r \in \Q_R \mid r < x\}.\]

Let \(n\) be a positive integer such that \(n > x\). Then \(\varphi(n)\) is an upper bound of the set \(\varphi(A_x)\). We can then extend \(\varphi\) to a map \(R \rightarrow S\) by defining \(\varphi(x)\) as the supremum of \(\varphi(A_x)\), which exists since \(S\) is complete. We then have to prove that \(\varphi\) preserves order, is bijective and is a homomorphism.

  1. \(\varphi\) preserves order

Suppose \(x,x'\) are elements of \(R\) such that \(x < x'\). Then there is a rational \(r\) such that \(x < r < x'\). In other words, \(r \in A_{x'}\) and \(r > a\) for all \(a \in A_x\). Taking images under \(\varphi\), we see that \(\varphi(r) > b\) for all \(b \in \varphi(A_x)\), so \(\varphi(r)\) is an upper bound for \(\varphi(A_x)\). In particular, \(\varphi(r) \ge \varphi(x)\).

On the other hand, since \(\varphi(r) \in \varphi(A_{x'})\), we have \(\varphi(r) \le \sup \varphi(A_{x'}) = \varphi(x')\). In fact, the inequality is strict: if equality holds, then \(\varphi(x') = \varphi(r) \in \Q_S\); by injectivity of \(\varphi\) this implies \(x' = r\), which contradicts the inequality in the previous paragraph. Thus \(\varphi(x) \le \varphi(r) < \varphi(x')\).

  1. \(\varphi\) is bijective

Injectivity follows easily from the fact that \(\varphi\) preserves order, so it suffices to show surjectivity. Let \(y\) be an element of \(S\). Then \(y\) is the supremum of the set \[B_y = \{s \in \Q_S \mid s < y\}.\]

There exists a positive integer \(m\) such that \(m > y\), so \(B_y\) is bounded above by an element of \(\Q_S\). Taking preimages, we have that \(\varphi^{-1}(B_y)\) is bounded above by \(\varphi^{-1}(m)\) (since \(\varphi\) preserves order), so it has a supremum \(x\). We claim that \(\varphi(x) = y\).

If \(a \in \varphi^{-1}(B_y)\), then \(a \le x\), which implies \(\varphi(a) \le \varphi(x)\). Hence \(\varphi(x)\) is an upper bound of \(B_y\), so \(y \le \varphi(x)\). However, if \(y < \varphi(x)\), then there is some rational \(s \in \Q_S\) such that \(y < s < \varphi(x)\). Taking preimages yields \(\varphi^{-1}(s) < x\), which means that \(\varphi^{-1}(s)\) is not an upper bound for \(\varphi^{-1}(B_y)\). In particular, there is some \(r \in \varphi^{-1}(B_y)\) such that \(r > \varphi^{-1}(s)\). But then we have \(\varphi(r) > s > y\), which is a contradiction since \(\varphi(r) \in B_y\).

  1. \(\varphi\) is a homomorphism

We begin by proving the following lemmas:

Let \(x,x'\) be elements of a complete ordered field \(R\), and use the above notation for \(A_x,A_{x'}\). Then \[x+x' = \sup \{r+s \mid r \in A_x, s \in A_{x'}\}.\]

Furthermore, if \(x,x' > 0\), then \[xx' = \sup \{rs \mid r,s > 0, r \in A_x, s \in A_{x'}\}.\]

  1. Clearly \(x+x'\) is an upper bound. Suppose \(t\) is another upper bound. Then \(t \ge r+s\) for all \(r \in A_x\), \(s \in A_{x'}\). Thus \(t-r\) is an upper bound for \(A_{x'}\), so \(t-r \ge x'\). This then implies that \(t-x'\) is an upper bound for \(A_x\), so \(t-x' \ge x\). Thus \(t \ge x+x'\).
  2. The argument is essentially the same as 1.

Let \(A\) be a non-empty subset of \(R\) that is bounded above, and let \(\varphi\) be the map defined above. Then \[\varphi(\sup A) = \sup \varphi(A).\]

For all \(a \in A\), we have \(\sup A \ge a\), so \(\varphi(\sup A) \ge \varphi(a)\) which shows \(\varphi(\sup A)\) is an upper bound. Suppose \(x\) is another upper bound of \(\varphi(A)\). Then \(x \ge \varphi(a)\) for all \(a \in A\). Taking preimages yields \(\varphi^{-1}(x) \ge a\), which implies \(\varphi^{-1}(x) \ge \sup A\). Thus \(x \ge \varphi(\sup A)\). This completes the proof.

From the lemmas, we have

\begin{align*} \varphi(x+x') &= \varphi(\sup \{r+s \mid r \in A_x, s \in A_{x'}\}) \\ &= \sup \varphi(\{r+s \mid r \in A_x, s \in A_{x'}\}) \\ &= \sup \{t+u \mid t \in \varphi(A_x), s \in \varphi(A_{x'})\} \\ &= \sup \{t+u \mid t \in B_{\varphi(x)}, s \in B_{\varphi(x')}\} \\ &= \sup B_{\varphi(x)} + \sup B_{\varphi(x')} \\ &= \varphi(x) + \varphi(x'). \end{align*}

Only the fourth equality requires extra justification: if \(r \in \Q_R\) and \(r < x\), then \(\varphi(r) \in \Q_S\) and \(\varphi(r) < \varphi(x)\), so that \(\varphi(A_x) \subseteq B_{\varphi(x)}\). Conversely, if \(s < \varphi(x)\), then \(s = \varphi(r')\) for some \(r'\) where \(r' < x\), so that \(B_{\varphi(x)} \subseteq \varphi(A_x)\). The same argument can be used to show that \(\varphi(xx') = \varphi(x)\varphi(x')\) for positive \(x,x'\).

Lastly, the multiplicativity of \(\varphi\) can be shown for negative numbers. Supposing \(x,x' > 0\) as before, we have \(f(-x)f(x') + f(x)f(x') = 0\), so \[f(-x)f(x') = -f(x)f(x') = -f(xx') = f(-xx').\] Likewise since \(f(-x)f(-x') + f(x)f(-x') = 0\), it follows that \[f(-x)f(-x') = -f(x)f(-x') = -f(-xx') = f(xx').\]

1.3.1: Classification of intervals in \(\R\)

Prove that every interval in \(\R\) is one of the following nine types:

\begin{align*} &(-\infty,\infty), (a,\infty), [a,\infty), \\ &(-\infty,b), (-\infty,b], (a,b), \\ &[a,b), (a,b], [a,b]. \end{align*}

Let \(I\) be an interval. We shall consider multiple cases:

  1. \(I\) not bounded above or below

    For any real number \(r\), there are real numbers \(s,t \in I\) such that \(s < r < t\). Thus \(r \in I\) as well, so \(I = (-\infty,\infty)\).

  2. \(I\) not bounded below, bounded above

    Since \(I\) is bounded above, it has a supremum \(b\). Let \(r\) be a real number such that \(r < b\). Then there must be some \(t \in I\) such that \(r < t \le b\), otherwise \(r\) would be an upper bound. Since \(I\) is not bounded below, there is also some \(s \in I\) such that \(s < r\). Since \(s < r < t\), we have \(r \in I\).

    If \(b \in I\), then \(I = (-\infty,b]\). Otherwise, \(I = (-\infty,b)\).

  3. \(I\) not bounded above, bounded below

    Similarly, \(I\) can be classified as either \((a,\infty)\) or \([a,\infty)\), depending on whether \(a = \inf I \in I\).

  4. \(I\) bounded below and above

    \(I\) has an infimum \(a\) and a supremum \(b\). If \(a = b\), then by 1.2.3 \(I\) contains only one point, which is obviously untrue. Now suppose \(a < r < b\). There must be some \(s \in I\) such that \(a \le s < r\), otherwise \(r\) is a lower bound for \(I\). Likewise, there is some \(t \in I\) such that \(r < t \le b\). Since \(s < r < t\), it follows that \(r \in I\).

    If \(a \in I\), then either \(I = [a,b)\) or \(I = [a,b]\) depending on whether \(b \in I\). Otherwise, either \(I = (a,b)\) or \(I = (a,b]\).

1.3.3: Intersection of all closed intervals containing a bounded real subset

Let \(A\) be a bounded subset of \(\R\). What is the intersection of all closed intervals which contain \(A\)?

Let \(B = [\inf A, \sup A]\), let \(S\) be the intersection, and let \(I\) be a closed interval containing \(A\) (note that \(I = [\inf I, \sup I]\)). We have \(\inf I \le \inf A\); if \(\inf I > \inf A\) then there must exist some \(a \in A\) such that \(\inf I > a \ge \inf A\), implying that \(a \not\in I\), a contradiction. Likewise, we have \(\sup A \le \sup I\), so that \(B \subseteq I\), and so \(B \subseteq S\). Furthermore, it is clear that \(B\) is a closed interval containing \(A\), so that \(S \subseteq B\). Thus \(S = I\).

1.3.4: Intersections of nested intervals

Give an example of a nested sequence of open intervals for which the intersection is empty. Give similar examples for which the intersection is a set with one member, an open interval, a closed interval, a semi-closed interval.

Let \(n\) run through the natural numbers. Then

\begin{align*} \bigcap_n\ \left(0,\frac{1}{n}\right) &= \varnothing, \\ \bigcap_n\ \left(-\frac{1}{n},\frac{1}{n}\right) &= \{0\}, \\ \bigcap_n\ (-1,1) &= (-1,1), \\ \bigcap_n\ \left(-1-\frac{1}{n},1+\frac{1}{n}\right) &= [-1,1], \\ \bigcap_n\ \left(0,1+\frac{1}{n}\right) &= (0,1]. \end{align*}

1.3.6: Nested closed intervals in \(\Q\) with empty intersection

Suppose you were working in the rational number system. Describe a nested sequence of closed intervals for which the intersection is empty.

We will first in work in \(\R\). Let \(x_1 \le x_2 \le \ldots\) be a non-decreasing sequence of rational numbers such that \(\sup \{x_i\} = \sqrt{2}\). Likewise, let \(y_1 \ge y_2 \ge \ldots\) be a non-increasing sequence of rational numbers such that \(\inf \{y_i\} = \sqrt{2}\). Interpreting the interval \([x_i,y_i]\) as the set \(\{r \in \Q \mid x_i \le r \le y_i\}\), we have that \(\cup_i [x_i,y_i] = \varnothing\), since \(\sqrt{2} \not\in \Q\).
