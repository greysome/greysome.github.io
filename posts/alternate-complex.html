An alternate construction of the complex numbers

This post does not assume much more than basic knowledge of set theory and complex numbers, as well as familiarity with the concept of remainders in arithmetic (when dividing one number by another).

Introduction

The standard definition of complex numbers establishes the value \(i = \sqrt{-1}\) and then constructs the set \[\{a+bi \mid a,b \in \R\},\] along with an addition and multiplication operation. It seems reasonable enough, yet it still raises a few quesions: does it really make sense to define a number \(i\) by the property that \(i^2 = -1\)? Why do we define addition and multiplication this way, and not another way? An alternate construction may help clarify these issues.

Quotient rings of \(\Z\)

The central concept being introduced here is that of a quotient ring. Before we talk about quotient rings, we must first talk about rings.

Roughly speaking, they are sets equipped with an addition and multiplication operation which satisfy a few properties; these properties will not be mentioned here. The main fact to keep in mind is that the sets \(\Z,\Q\) and \(\R\) are rings under the standard addition and multiplication, and the set of rational polynomials \(\Q[x]\) as well as the set of real polynomials \(\R[x]\) are rings under polynomial addition and multiplication.

In \(\Z\), there is the notion of modular arithmetic, where addition and multiplication is done \(\text{mod}\) a particular number (say 4). For instance,

\begin{align*} 2 + 3 &\equiv 1\ (\text{mod}\ 4) \\ 3 \times 5 &\equiv 3\ (\text{mod}\ 4). \end{align*}

The statement \(a \equiv b\ (\text{mod}\ 4)\) means "\(a\) and \(b\) have the same remainder \(\text{mod}\ 4\)".

We could try to define new operations \(+_4\) and \(\times_4\) (shorthand for "addition/multiplication \(\text{mod}\ 4\)"), such that the congruences become equalities:

\begin{align*} 2 +_4 3 &= 1 \\ 3 \times_4 5 &= 3 \end{align*}

However, this introduces some ambiguity. For example, the expression \(2 +_4 3 = 5\) is valid as well, since \(2 + 3 \equiv 5\ (\text{mod}\ 4)\). Yet clearly \(1 \neq 5\) as integers.

What we would like is a way to treat numbers with the same remainder \(\text{mod}\ 4\) as "being equal", so that the operations \(+_4\) and \(\times_4\) become well-defined. Consider the sets

\begin{align*} \bar{0} &= \{\cdots,-8,-4,0,4,8,\cdots\} \\ \bar{1} &= \{\cdots,-7,-3,1,5,9,\cdots\} \\ \bar{2} &= \{\cdots,-6,-2,2,6,10,\cdots\} \\ \bar{3} &= \{\cdots,-5,-1,3,7,11,\cdots\} \end{align*}

of integers with the same remainder \(\text{mod}\ 4\) (note that \(0,1,2,3\) are all the possible remainders \(\text{mod}\ 4\)). The key insight is to consider the sets themselves as individual elements, and then define \(+_4\) and \(\times_4\) on these elements. Namely,

\begin{align*} \bar{i} +_4 \bar{j} &= \text{the set containing $i+j$} \\ \bar{i} \times_4 \bar{j} &= \text{the set containing $ij$}. \end{align*}

We have \(\bar{2} +_4 \bar{3} = \bar{1}\), because \(\bar{1}\) contains the element \(2+3 = 5\). Similarly \(\bar{2} \times_4 \bar{3} = \bar{2}\).

It turns out that \(+_4\) and \(\times_4\) satisfy the properties required of a ring. Hence, the elements \(\{\bar{0}, \bar{1}, \bar{2}, \bar{3}\}\) under these operations form a ring called \(\Z/4\Z\). This construction can be generalised, giving rise to a family of rings \(\Z/n\Z\) where \(n\) is a positive integer. The name "quotient ring" reflects the choice of notation, which looks like a fraction \(\frac{\Z}{n\Z}\).

Quotient rings of \(\R[x]\)

Earlier, we constructed the sets \(\bar{i}\) of numbers with the same remainder \(\text{mod}\ 4\), defined the operations \(+_4\) and \(\times_4\) on these sets, and showed that this forms a ring.

The general procedure works not just on integers but on polynomials as well. But we first have to establish what it means by "taking remainders mod a polynomial".

Fix a real polynomial \(r(x)\), and let \(p(x),q(x)\) be arbitrary real polynomials. When we say that \(p(x) \equiv q(x)\ (\text{mod}\ r(x))\), we mean that \[p(x) = c(x)r(x)+q(x)\] for some polynomial \(c(x)\). In other words, \(p(x)\) and \(q(x)\) differ by a polynomial multiple of \(r(x)\).

We can say that \(x^2+4x+2 \equiv x\ (\text{mod}\ x+1)\), because \[x^2+4x+2 = (x+2)(x+1) + x.\]

This is analagous to the integers: if \(a \equiv b\ \text{mod}\ n\), that means they differ by some multiple of \(n\).

The remainder of a polynomial \(p(x)\) \(\text{mod}\ r(x)\) is the smallest degree polynomial \(q(x)\) such that \[p(x) \equiv q(x)\ (\text{mod}\ r(x)).\]

As mentioned above, \[x^2+4x+2 \equiv x\ (\text{mod}\ x+1),\] but it is also true that \[x^2+4x+2 \equiv x^2+3x+1\ (\text{mod}\ x+1).\] (Why?) However, \(x\) is the smallest degree polynomial \(p(x)\) satisfying \(x^2+4x+2 \equiv p(x)\ (\text{mod}\ x+1)\), so it is the remainder of \(x^2+4x+2\ \text{mod}\ x+1\).

Note that if \(q(x)\) is the remainder of \(p(x)\ \text{mod}\ r(x)\), then \(\deg q \le \deg r\). This fact is important, as we shall see.

We shall proceed to construct a quotient ring of \(\R[x]\), just as we did with the integers. Let \(r(x) = x^2\). The set \[\{ax+b \mid a,b \in \R\}\] is precisely the set of all possible remainders \(\text{mod}\ x^2\). For each \(a,b \in \R\), we shall define the set \(\overline{ax+b}\) to be the set of all polynomials whose remainder \(\text{mod}\ x^2\) is \(ax+b\). Just as before, we let

\begin{align*} \overline{ax+b} +_{x^2} \overline{cx+d} &= \text{the set containing $(ax+b)+(cx+d)$} \\ \overline{ax+b} \times_{x^2} \overline{cx+d} &= \text{the set containing $(ax+b)(cx+d)$}. \end{align*}

The operations \(+_{x^2}\) and \(\times_{x^2}\) do satisfy the properties required of a ring, so the elements \(\overline{ax+b}\) under these operations form a ring called \(\R[x]/(x^2)\).

We could replace \(x^2\) with any polynomial \(r(x)\), which would induce a corresponding quotient ring \(\R[x]/(r(x))\). The elements of this quotient ring are of the form \(\overline{q(x)}\), where \(q(x)\) is any polynomial satisfying \(\deg q \le \deg r\).

The alternate construction

Here's the amazing fact: we could simply define the complex numbers \(\C\) to be the quotient ring \(\R[x]/(x^2+1)\)! What do polynomials have to do with complex numbers? The relationship is surprising:

  1. The element \(\bar{x}\) satisfies \(\bar{x}^2 + \bar{1} = \bar{0}\), just like how \(i^2+1 = 0\).
  2. Each element of \(\R[x]/(x^2+1)\) is of the form \(\overline{ax+b}\) (or equivalently \(\overline{b+ax}\)), just like how each complex number is of the form \(p+qi\) for real \(p,q\).
  3. It can be shown that \[\overline{b+ax} +_{x^2+1} \overline{d+cx} = \overline{(b+d)+(a+c)x}\] and that \[\overline{b+ax} \times_{x^2+1} \overline{d+cx} = \overline{(bd-ac)+(ad+bc)x}.\] (Does this seem familiar?)

Suppose we removed all the overlines, replaced all instances of \(x\) with \(i\), and interpreted the operations \(+_{x^2+1}\) and \(\times_{x^2+1}\) as complex addition/multiplication instead. What we get is indistinguishable from the complex numbers! Hence, we might as well pretend they are in fact complex numbers (in more precise terms, there is a canonical isomorphism from \(\R[x]/(x^2+1)\) to \(\C\) given by \(\overline{ax+b} \mapsto b+ai\)).

Note how complex addition/multiplication is in a sense just polynomial addition/multiplication. Also notice how we did not prove the existence of an element \(x\) satisfying \(x^2+1 = 0\), so much as construct a new ring which happens to contain this element.

Closing remarks

This construction may seem like nothing more than a neat trick, but it really is fundamental to field theory. One would often start with the set of rationals \(\Q\), "add in" roots of polynomials, and then study the resulting set.

These roots of polynomials nearly always do not lie in \(\Q\) (for example, consider the polynomial \(x^2-2\)). By taking quotients of \(\Q[x]\), we are able to construct a bigger ring in which these roots exist, without having to assume their existence beforehand.

For instance, we know in the quotient ring \(\Q[x]/(x^2-2)\), the element \(\bar{x}\) satisfies \(\bar{x}^2-\bar{2} = \bar{0}\), i.e. \(\bar{x}\) is a root of the polynomial \(p(t) = t^2-2\). However, note how we did not have to assume the existence of \(\R\) (which in particular contains the roots \(\pm\sqrt{2}\)) to show that this root exists.
