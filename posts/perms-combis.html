A group-theoretic approach to basic counting

This post assumes basic knowledge on permutations and combinations, as well as group theory (particularly the symmetric group \(S_n\), the orbit-stabilizer theorem, and Burnside's lemma).

Introduction

The goal is to provide a group-theoretic framework for the topic of counting. The key idea is to encode symmetries of $k$-permutations as group actions, and then use Burnside's lemma to count the number of distinct objects under these symmetries.

Everything goes back to permutations

The fundamental objects of study are $k$-permutations. All other structures should be thought of as equivalence classes of $k$-permutations under some symmetry. Here are some examples (let \(x,y\) be arbitrary $k$-permutations of some set \(S\)):

Structure \(x \sim y\)
combination \(x\) contains precisely the same elements as \(y\)
labelled permutation1 the sequence of labels in \(x\) and \(y\) are equal
necklace \(x\) is a shifted version of \(y\)

An elementary result is first established before we move on:

Let \(S\) be a finite set with \(n\) elements. The number of $k$-permutations of \(S\), denoted \(\perm{n}{k}\), is \(\frac{n!}{(n-k)!}\).

The simplest method involves multiplying the number of possibilities for the 1st entry, and the 2nd entry, and so on, yielding a product \[n(n-1)\cdots(n-(k-1)) = \frac{n!}{(n-k)!}.\]

There is an alternative approach using group actions.

Let \(A\) be the set of all $k$-permutations of \(S\). Let \(S_n\) act on \(A\) by entry-wise evaluation. Clearly this action is transitive, as there is a bijection of \(S\) mapping any $k$-permutation to any other $k$-permutation. Hence, it suffices to consider the orbit of one $k$-permutation, say \(x = (1,2,\cdots,k)\).

The stabilizer of \(x\) is the group of bijections fixing \(1,2,\cdots,k\), which is precisely \(S_{n-k}\). Thus by the orbit-stabilizer theorem, \[ \perm{n}{k} = \left|\mathcal{O}_x\right| = \frac{\left|S_n\right|}{\left|S_{n-k}\right|} = \frac{n!}{(n-k)!}. \]

In the above proof, each element of \(S\) was identified with a unique number from \(1\) to \(n\). In future discussion, we will do likewise and work with these numbers directly. Also, from now on \(A\) refers to the set defined in the proof.

Symmetries of $k$-permutations as group actions

As mentioned earlier, the structures mentioned in the above table should be treated as equivalence classes of $k$-permutations. Equivalently, they are the orbits of specific group actions:

Let \(x\) be an arbitrary $k$-permutation.

  1. Let \(S_k\) act on \(A\) by permuting the entries in each $k$-permutation. Then the orbit of \(x\) consists of $k$-permutations containing the entries of \(x\), but listed in different orders. This orbit can be identified with a $k$-combination, where order does not matter.

  2. A label symmetry of a labelled set \(S\) is defined to be a label-preserving bijection. It can be seen that the group of label symmetries \(L\) is the direct product \(\prod_i S_{n_i}\) (see 1 regarding notation).

    Let \(L\) act on \(A\) by entry-wise evaluation. Then the orbit of \(x\) contains all $k$-permutations that are equal to \(x\) up to label symmetry. This orbit can be identified with a labelled $k$-permutation.

  3. Let \(C_k\) act on \(A\) by shifting each entry a fixed number of places. The orbit of \(x\) can be identified with a $k$-necklace.

By adopting the language of group actions, we can apply Burnside's lemma to calculate the number of orbits.

Let \(G\) be a group acting on the set \(X\). Then the number of orbits \(\left|X/G\right|\) is given by \[\left|X/G\right| = \frac{1}{\left|G\right|} \sum_{g \in G} \left|X^g\right|,\] where \[X^g = \{x \in X \mid g \cdot x = x\}.\]

Each point is a continuation from the previous example. Let \(g\) be an arbitrary element in the respective group. Clearly if \(g = 1\) then \(\left|A^g\right| = \perm{n}{k}\).

  1. If \(g\) is nontrivial, then \(A^g\) is empty. Thus the number of $k$-combinations is \[\frac{\left|A^1\right|}{\left|S_k\right|} = \frac{n!}{k!(n-k)!}.\]

  2. For nontrivial \(g\), let \(M_g\) be the subset of \(S\) not fixed by \(g\) and let \(d_g = \left|M_g\right|\). Then \(\left|A^g\right|\) is the number of permutations which are disjoint from \(M_g\), namely \(\perm{n-d_g}{k}\) (we define \(\perm{d}{k} = 0\) if \(d < k\)). So the number of labelled $k$-permutations is \[\frac{1}{\left|L\right|} \sum_{g \in L} \perm{n-d_g}{k}.\]

    In the special case \(k=n\), \(\perm{n-d_g}{k} = 0\) for all nontrivial \(g\), since \(n-d_g < k\). Thus the number of labelled $n$-permutations is \[\frac{\perm{n-d_1}{n}}{\left|L\right|} = \frac{n!}{n_1!n_2! \cdots n_m!}.\]

  1. Using similar reasoning as above, the number of $k$-necklaces is \[\frac{\left|A^1\right|}{\left|C_k\right|} = \frac{n!}{k(n-k)!}.\]

Combining symmetries

Consider the following problem: a basket contains three apples, two oranges and two bananas. Suppose three fruits were selected from this basket without replacement; say that selections \(A\) and \(B\) are equal if the number of each type of fruit in \(A\) and \(B\) are equal. How many possible selections are there?

Just as above, this question involves counting the number of objects under some symmetry. But this time, the symmetry in question is actually a combination of two symmetries: namely permutation and label symmetry. The order in which the fruits were selected does not matter, and only the type of fruit is considered when determining when two selections are equal.

Let \(G\) and \(H\) be two symmetry groups. In general, the 'combined' symmetry group takes the form of a semidirect product \(G \rtimes H\) or \(H \rtimes G\). More details can be found here.

  1. The relevant symmetry groups for the problem above are \(S_3\) and \(L = S_3 \times S_2 \times S_2\). Note that for all \(p \in S_3\) and \(l \in L\), the actions of \(p\) and \(l\) on a $3$-permutation commute with each other. Hence the combined symmetry group is simply the direct product \(S_3 \times L\), and the problem can be solved by applying Burnside's lemma to this group.

  2. Let \(x = (a,b,c,d)\) be a $4$-permutation, and arrange the entries in a square:

    Let the nontrivial element of the group \(C_2\) act on this square by flipping it horizontally. Combining this flip symmetry with shift symmetry (in the case of this square, rotational symmetry) yields a semidirect product \[C_4 \rtimes C_2 \cong D_8.\]

Footnotes:

1

For our purposes, we shall define a labelled set as a set \(S\) with labels \(l_1,l_2,\cdots,l_m\), such that each \(s \in S\) is associated with a unique \(l_i\). The multiplicity of \(l_i\) is denoted by \(n_i\). A labelled permutation is a permutation of \(S\), except two labelled permutations are equal if the corresponding sequences of labels are equal.
