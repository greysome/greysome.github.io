Identifying finite fields

The prime subfield of a finite field \(F\) is isomorphic to \(\F_p\) for some prime \(p\). Thus we can view \(F\) as a vector space over \(\F_p\): supposing the dimension of the vector space is \(n\), let \(\{x_1, \cdots, x_n\}\) be any basis. Then every element of \(F\) can be uniquely represented in the form \[\sum_{i=1} a_i x_i,\] where \(a_i \in \F_p\). This proves that \(F\) must have \(p^n\) elements.

Conversely, we can show that for any prime \(p\) and positive integer \(n\) there exists a finite field of order \(p^n\) by explicitly constructing one: namely the splitting field of the polynomial \(f(x) = x^{p^n}-x\) over \(\F_p\). It is separable as the derivative \(f'(x) = {p^n}x^{p^n-1}-1 = -1\) has no roots, meaning there are \(p^n\) distinct roots. We can then show that these \(p^n\) roots form a field, by using the fact that in characteristic \(p\), we have \((\alpha+\beta)^{p^n} = \alpha^{p^n}+\beta^{p^n}\) and \((\alpha\beta)^{p^n} = \alpha^{p^n}\beta^{p^n}\). In particular, the prime subfield \(\F_p\) is contained in the roots since \(1\) is a root. Thus the splitting field has precisely \(p^n\) elements.

Furthermore, we can show that finite fields of order \(p^n\) are unique up to isomorphism. Suppose the finite field \(F\) has dimension \(n\) over its prime subfield \(\F_p\). Its multiplicative group \(F^\times\) is cyclic and has order \(p^n-1\), so for \(a \neq 0\) we have \(a^{p^n} = a\). Clearly \(0^{p^n} = 0\) as well, so the elements of \(F\) are all roots of the polynomial \(f(x) = x^{p^n}-x \in \F_p[x]\). But there are at most \(\deg f = p^n\) roots, and there are precisely \(p^n\) elements in \(F\). Thus \(F\) is the splitting field of \(f(x)\). It follows then that finite fields of order \(p^n\) are unique up to isomorphism since splitting fields are unique up to isomorphism.

Essentially, for every prime \(p\) and positive integer \(n\), there is one and only one finite field with order \(p^n\), namely the splitting field of the polynomial \(x^{p^n}-x\). Thus we can unambiguously denote the field \(\F_{p^n}\). NOTE: \(\F_{p^n}\), just like \(\F_p\), has characteristic \(p\).

Irreducible polynomials in \(\F_p[x]\)

We can proceed to prove the existence of irreducible polynomials over \(\F_p\) of all degrees, and derive a formula which will allow us to find explicitly all the irreducible polynomials over \(\F_p\).

If \(\theta\) is a generator of the cyclic multiplicative group \(\F_{p^n}^\times\), then we can write \(\F_{p^n} = \F_p(\theta)\). Since \(\F_{p^n}\) has degree \(n\) over \(\F_p\), the minimal polynomial \(m_\theta(x)\) has degree \(n\). Minimal polynomials are irreducible, so we have essentially shown that degree \(n\) irreducible polynomials exist, where \(n\) is any positive integer.

This gives us another way to construct \(\F_{p^n}\) as the quotient \(\F_p[x]/(f(x))\), where \(f\) is an irreducible degree \(n\) polynomial. Being able to express elements of \(\F_{p^n}\) as residue classes of polynomials is useful for carrying out concrete computations.

A particularly significant result is that if \(f(x)\) is an irreducible polynomial of degree \(n\) over \(\F_p\), then \(f(x) \mid x^{p^n}-x\). If \(\alpha\) is a root of \(f(x)\), then \(\F_p(\alpha)\) is a degree \(n\) extension of \(\F_p\) and is thus isomorphic to \(\F_{p^n}\). In particular, \(\alpha^{p^n} = \alpha\) so it is a root of \(x^{p^n}-x\).

A related result is that \(d \mid n\) if and only if \(x^{p^d}-x \mid x^{p^n}-x\). The forward implication follows from the fact that \(x^{p^d-1}-1 \mid x^{p^n-1}-1\), since \(p^d-1 \mid p^n-1\). As for the reverse implication, we can use the division algorithm to write \(n = qd+r\) where \(0 \le r < d\). Let \(f(x)\) be an irreducible degree \(d\) polynomial with a root \(\alpha\). Since \(\alpha^{p^d} = \alpha\), we have a chain of equalities \(\alpha = \alpha^{p^{2d}} = \cdots = \alpha^{p^{qd}}\); multiplying the first and last times by both sides gives \(\alpha^{p^r} = \alpha^{p^n}\). Also, \(\alpha\) is a root of \(x^{p^n}-x\), so \(\alpha^{p^n} = \alpha\). Thus \(\alpha^{p^r} = \alpha\). Now we shall identify \(\alpha \in \F_p(\alpha)\) with the element \(\bar{x} \in \F_p[x]/(f(x))\). It is known that \(g(\bar{x})^{p^r} = g(\bar{x}^{p^r}) = g(\bar{x})\) for every polynomial \(g(x) \in \F_p[x]\). Since every congruence class has a representative of the form \(g(\bar{x})\), it follows that every element of \(\F_p[x]/(f(x))\) is a root of the polynomial \(X^{p^r}-X\) in \(X\). If \(r > 0\), then \(X^{p^r}-X\) has nonzero degree and has at most \(p^d\) roots, so \(p^d \le p^r\), a contradiction since \(0 \le r < d\). Thus \(r = 0\) and \(d \mid n\).

The next step is showing that each irreducible polynomial dividing \(x^{p^n}-x\) only appears once in the factorization of \(x^{p^n}-x\). This is easy to see since \(x^{p^n}-x\) is separable.

By combining the past few results, we can conclude that \[ x^{p^n}-x = \prod_{d \mid n} \prod_{\substack{f\ \text{irreducible} \\ \deg f = d}} f(x). \]

Using this result, we can recursively compute all degree \(n\) polynomials. Furthermore, we can derive a formula for the number of degree \(n\) irreducible polynomials \(\phi(n)\) using the Mobius inversion formula: \[ \phi(n) = \frac{1}{n} \sum_{d \mid n} \mu(d)p^{n/d}, \] where \(\mu(n)\) is the Mobius $μ$-function.

The Galois group \(\text{Gal}(\F_{p^n}/\F_p)\)

As mentioned earlier, \(\F_{p^n}\) is the splitting field of the separable polynomial \(x^{p^n}-x\) is hence Galois. It can be verified that the map \(\sigma_p : \F_{p^n} \rightarrow \F_{p^n}\) given by \(x \mapsto x^p\) is an automorphism. In \(G = \text{Gal}(\F_{p^n}/\F_p)\), it generates an order \(n\) subgroup and the elements are \(\sigma_p^i\) where \(0 \le i < n\). But by the Fundamental Theorem of Galois theory, \(\left|G\right| = [\F_{p^n} : \F_p] = n\). Thus \(G \cong Z_n\) with \(\sigma_p\) as a generator.

The subfields of \(\F_{p^n}\) containing \(\F_p\) correspond to the subgroups of the cyclic group \(Z_n\), which in turn correspond to the divisors of \(n\). Concretely, for every divisor \(d\) of \(n\), \(Z_d\) is a subgroup of \(Z_n\) with corresponding fixed field \(\F_{p^{n/d}}\): it suffices to check that \(\F_{p^{n/d}}\) is the fixed field of \(\sigma_p^{n/d}\). Clearly all the elements of \(\F_{p^{n/d}}\) are roots of the polynomial \(x^{p^{n/d}}-x\), and there are at most \(p^{n/d}\) such roots, giving the result. This correspondence between subfields and divisors implies that \(\F_{p^d} \subseteq \F_{p^n}\) if and only if \(d \mid n\).

We can also consider the Galois conjugates of some \(\alpha \in \F_{p^n} \setminus \F_p\) (note that if \(\alpha \in \F_p\) then it is fixed by all automorphisms, so that the only Galois conjugate is itself). Since the elements of \(\text{Gal}(\F_{p^n}/\F_p)\) are of the form \(\sigma_p^i\) where \(0 \le i < n\), it follows that the Galois conjugates are \(\alpha, \alpha^p, \alpha^{p^2}, \cdots, \alpha^{p^{n-1}}\). We can further conclude that these $p$-powers of \(\alpha\) are precisely the roots of \(m_\alpha(x)\), since each \(\sigma_p^i\) must send roots to roots.
