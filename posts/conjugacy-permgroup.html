Conjugacy in \(S_n\)

Let \(\sigma,\tau \in S_n\). We shall let \(\sigma^\tau\) denote the conjugate \(\tau\sigma\tau^{-1}\). If the cycle decomposition of \(\sigma\) is given by \[(a_1\ a_2\ \ldots\ a_i)(b_1\ b_2\ \ldots\ b_j) \ldots,\]

then the cycle decomposition of \(\sigma^\tau\) is \[(\tau a_1\ \tau a_2\ \ldots\ \tau a_i)(\tau b_1\ \tau b_2\ \ldots\ \tau b_j) \ldots;\]

which is to say, the elements appearing in the cycle decomposition are permuted according to \(\tau\). In particular, the cycle type is the same. Conversely, for each permutation \(\pi\) with the same cycle type as \(\sigma\), there is clearly a \(\tau\) such that \(\sigma^\tau = \pi\). This shows:

The conjugacy classes in \(S_n\) consist of all permutations with the same cycle type.

Consider the action of \(S_n\) on itself by conjugation. Let the cycle type of \(\sigma\) be \((n_1^{k_1},n_2^{k_2},\ldots,n_i^{k_i})\). The stabiliser of \(\sigma\) consists of those elements which permute the cycle elements in a cyclic fashion, and/or permute the \(k_i\) cycles of equal length amongst themselves. Thus the stabiliser is the product \[\prod_i C_{n_i} \wr S_{k_i},\]

which has order \[\prod_i n_i^{k_i} k_i!\]

Since the orbit of \(\sigma\) is its conjugacy class, the number of elements with cycle type \((n_1^{k_1},n_2^{k_2},\ldots,n_i^{k_i})\) is \[\label{eq1}\tag{*} \frac{n!}{\prod_i n_i^{k_i} k_i!}.\]

Conjugacy in \(A_n\)

Unlike \(S_n\), the conjugacy classes in \(A_n\) do not necessarily correspond to cycle type. This is because two permutations may not be conjugate to each other by an even permutation. However, with a bit of work we can still determine the conjugacy classes.

Let \(\sigma \in A_n\). Since \(A_n \triangleleft S_n\) and \(S_n\) acts transitively on the conjugacy class of \(\sigma\), the orbits under the action of \(A_n\) have equal size, and so they must divide the orbit size under \(S_n\). In particular, \[\frac{|S_n|}{|C_{S_n}(\sigma)|} = k\frac{|A_n|}{|C_{A_n}(\sigma)|}\]

for some positive integer \(k\). Rearranging gives \[k = \frac{2}{|C_{S_n}(\sigma) : C_{A_n}(\sigma)|};\]

the only possibilities for the denominator are \(1\) and \(2\). Note that \(|C_{S_n}(\sigma) : C_{A_n}(\sigma)| = 1\) if and only if \(\sigma\) is not centralised by an odd permutation, and it turns out that

\(\sigma \in A_n\) is not centralised by an odd permutation if and only if its cycle type consists of distinct odd integers.

Thus, if the cycle type of \(\sigma\) consists of distinct odd integers, then the conjugacy class of \(\sigma\) in \(S_n\) splits into two equal-size conjugacy classes in \(A_n\). Otherwise, the conjugacy classes of \(\sigma\) in \(S_n\) and \(A_n\) are the same.

Example: \(A_5\) is not simple

The cycle types appearing in \(A_5\) are \[(1^5), (1^2,3), (1,2^2), (5^1);\]

the only one consisting of distinct odd integers is \((5^1)\). Using this fact and \(\eqref{eq1}\), the sizes of the conjugacy classes are \(1, 20, 15, 12\) and \(12\).

Now, any normal subgroup of \(A_5\) is a union of conjugacy classes, and so it must be a sum containing these numbers (note that the sum must contain \(1\)). However, besides the sums \(1\) and \(1+20+15+12+12\), no other sum divides \(60\), thus there are no non-trivial proper normal subgroups.
